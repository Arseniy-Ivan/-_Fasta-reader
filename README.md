# Общее о Fasta Reader
Автор программы: Захаров Иван. 
Программа Fasta Reader позволяет читать файлы формата Fasta, а также получать из них некоторую информацию, включая длину исследуемой последовательности и ее формат (нуклеотидная/белковая)
## Как установить?
1. Для использования FastaReader скачайте файл формата py с моего GitHub [Ссылочка](https://github.com/Arseniy-Ivan/-_Fasta-reader) .

2. Поместите его в той же дериктории, где находится ваш .fasta файл.

3. Откройте в программе, которая может работать с py файлами (например VSCode).

## Как использовать?
- При обычном запуске вам будет предложено ввести название файла или полный путь до него, если он не в той же папке, что FastaReader.py. После этого вам выведет информацию о нем.

- Если вам не нужно всё, то в поле «Менять выводные данные здесь:» можно поменять вывод. Обязательно прописать out. и сразу после, без пробела, одну из этих функций:
    - len() — Вычисляет длину последовательности и выводит результат.

    - str() — Приводит последовательность к виду строки.

    - seq_alphabet() — Определяет, какого последовательность типа (нуклеотидная/белковая).
